# Agent Democratization

## The Insight
Open-source agent frameworks are rapidly democratizing access to autonomous AI capabilities. The Claude Code leak's rapid replication by claw-code (100K stars in hours) demonstrated that harness-layer moats are narrow. Microsoft's Agent Framework v1.0, AgentScope, and CoPaw provide enterprise-grade agent infrastructure that rivals proprietary solutions.

## Evidence
- [[entities/claw-code]] — 181K+ stars in weeks; clean-room rewrite of Claude Code
- [[entities/openclaw]] — 302K stars; zero subscription model democratizing access
- [[topics/github_trends]] — Agent frameworks reaching production-grade maturity

## Implications
The competitive moat for agent frameworks is shifting from technical architecture to ecosystem integration, reliability, and enterprise support. This democratization accelerates adoption but also raises security concerns as less mature frameworks enter production.
