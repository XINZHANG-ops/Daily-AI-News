# OpenHands Software Agent SDK

## What It Is
OpenHands/software-agent-sdk is a clean, modular SDK for building AI agents that work with code. It supports one-off tasks, routine maintenance, and multi-agent operations. Can run locally or in ephemeral workspaces (Docker/Kubernetes). Powers OpenHands CLI and Cloud.

## Significance
OpenHands SDK represents the trend toward modular, production-grade agent frameworks. Unlike monolithic agent architectures, its clean separation of concerns (one-off tasks vs routine maintenance vs multi-agent) and deployment flexibility (local vs ephemeral workspaces) addresses different enterprise needs. The SDK's modularity suggests it's designed for composability — combining OpenHands with other tools rather than requiring a full platform lock-in.
