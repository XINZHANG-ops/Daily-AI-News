# Base MCP

## What It Is
Base MCP (Model Context Protocol) is Coinbase's extension of Anthropic's MCP protocol, enabling AI models like Claude and ChatGPT to transfer funds, swap tokens, check balances, and interact with DeFi protocols. Launched on May 27, 2026.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Launch Date | 2026-05-27 |
| Creator | Coinbase (Base) |
| Type | Protocol extension |
| Capabilities | Fund transfer, token swap, balance check, DeFi interaction |

## Significance
This is the missing link for autonomous AI agents handling financial transactions. MCP (Model Context Protocol) isn't just crypto-specific — it's the same protocol Anthropic released last year. Now Coinbase's Base extends it to DeFi, creating a standardized way for AI agents to move money.

The implication: autonomous AI agents can now execute real financial transactions, not just simulate them. This is the infrastructure layer for AI-to-AI commerce and AI handling real user funds. Base MCP enables:
- AI agents autonomously managing DeFi portfolios
- Cross-protocol arbitrage by AI agents
- Automated yield farming and lending decisions
- Real-time financial transaction execution without human approval
