# NemoClaw

## What It Is
NemoClaw is NVIDIA's reference stack for building physical AI agents, released as part of the NVIDIA Agent Toolkit on May 31, 2026. It provides blueprints and security patterns for deploying physical AI agents built on top of NVIDIA's Cosmos foundation models.

**Note**: There are two NemoClaw versions — this entry covers the May 2026 Agent Toolkit release. The March 2026 OpenClaw reference stack (NVIDIA/NemoClaw on GitHub with 1.1k stars) is a different project focused on securing OpenClaw agents.

## Significance
NemoClaw represents NVIDIA's approach to securing the physical AI agent deployment stack. Combined with OpenShell runtime and Cosmos 3 foundation models, NemoClaw provides the complete infrastructure for building, deploying, and securing physical AI agents in robotics and industrial applications.

The timing aligns with broader physical AI initiatives from Google (Gemini Robotics) and OpenAI (robotics hiring), positioning NVIDIA as the infrastructure provider for the emerging physical AI ecosystem.
