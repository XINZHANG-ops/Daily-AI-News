# NemoClaw

## What It Is
NemoClaw is NVIDIA's open-source reference stack for running OpenClaw securely with the NVIDIA OpenShell runtime. Released as an alpha preview in March 2026, it provides a hardened blueprint, state management, routed inference, and layered protection for running AI agents built on the OpenClaw architecture.
