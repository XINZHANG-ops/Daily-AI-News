# Spud

## What It Is
Spud is the internal codename for OpenAI's GPT-5.5 model family, referencing the model during its pretraining phase and subsequent enterprise pivot. The codename became public through OpenAI's strategic communications around its shift toward "high-value professional work" and super app integration. Spud represents the same model architecture later released commercially as GPT-5.5.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | OpenAI |
| Public Name | GPT-5.5 |
| Codename Origin | Internal pretraining designation |
| Target | High-value professional work, enterprise super app |

## Significance
OpenAI's explicit use of the "Spud" codename in public communications (e.g., "Spud model targeting high-value professional work") signals a marketing strategy: by separating the technical model from the branded product, OpenAI can position GPT-5.5 as a platform (the "super app") while Spud represents the raw capability underneath. The 49-day release cycle between Spud completion and GPT-5.5 deployment was reportedly designed for enterprise procurement lock-in — closing deals before procurement cycles end rather than competing on benchmarks alone.
