# Peek

## What It Is
Peek is an open-source database GUI that implements an infinite canvas interface, similar to Figma, for data exploration and visualization. It emphasizes collaborative data analysis through P2P (peer-to-peer) collaboration.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Type | Database GUI |
| Feature | Infinite Canvas / P2P Collaboration |
| Integration | MCP Support |
| Stars | 2.1k |

## Significance
Peek is significant for its adoption of the Model Context Protocol (MCP), allowing AI agents to interact directly with the database GUI. This enables a hybrid workflow where an agent can manipulate data visualizations or query structures on a spatial canvas, bridging the gap between traditional tabular data views and agentic interaction.
