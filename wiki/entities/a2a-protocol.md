# A2A Protocol

## What It Is
The Agent-to-Agent (A2A) Protocol is Google's protocol for inter-agent communication, launched April 2025 and celebrating its one-year anniversary in April 2026. It has grown to 150+ participating organizations and 22,000 GitHub stars. Now deployed in Azure AI Foundry and Amazon Bedrock AgentCore, A2A handles inter-agent communication across Microsoft, AWS, Salesforce, SAP, and ServiceNow. The v1.0 release introduced cryptographically signed Agent Cards for secure agent verification.

## Significance
A2A has emerged as a key standard for agent interoperability, with cross-vendor support from Microsoft, AWS, Salesforce, SAP, and ServiceNow. The cryptographically signed Agent Cards provide security for agent verification in production systems.
