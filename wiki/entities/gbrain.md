# gbrain

## What It Is
`garrytan/gbrain` is an open-source "Agent Brain" layer that provides synthesis and gap analysis through self-wiring knowledge graphs. It is designed to integrate with the Model Context Protocol (MCP) to enhance the capabilities of AI coding agents like Claude Code and Cursor.

## Significance
gbrain exemplifies the move toward "agentic infrastructure"—tools that don't just provide LLM access, but provide the cognitive architecture (like knowledge graphs) necessary for agents to perform complex, multi-step synthesis without losing context.
