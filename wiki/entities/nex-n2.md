# Nex-N2

## What It Is
Nex-N2 is an open-source agentic model family (Nex-N2-Pro and Nex-N2-mini) built on Qwen3.5 with a custom "Agentic Thinking" framework. Released June 7, 2026, it scores 80.8% on SWE-Bench Verified and 83.7% on BrowseComp, with a custom sglang fork for serving. The project signals the rising competitiveness of Qwen3.5-derived open agentic stacks.

## Significance
Nex-N2 represents the next phase of Qwen3.5's ecosystem expansion: community-built agentic stacks on top of the open-weight base, scoring within striking distance of frontier closed models. The 80.8% SWE-Bench Verified score is competitive with Claude Opus 4.7's published numbers — and it's free, open-weight, and uses a custom inference stack (sglang fork) optimized for agent serving. The "Agentic Thinking" framework is the project's differentiator: it treats agentic execution as a first-class design constraint, not an afterthought. As xAI's Grok Build 0.1 launches at $1/$2 per million tokens the same week, Nex-N2 demonstrates that the commodity open-weight frontier is closing the gap with closed frontier coding models — putting further margin pressure on Claude Code and GPT-5.4-Codex.
