# Long Horizon

## What It Is
An autonomous AI project builder that can create entire projects with self-looping execution and graph-connected memory that grows like a brain. Features a self-healing graph index for persistent knowledge.

## Significance
Long Horizon represents a novel architecture for autonomous agents: rather than single-pass execution, it uses self-looping to continuously build and extend projects. The graph-connected memory that grows like a brain is a distinct approach to persistent context, distinguishing it from vector-based memory systems.
