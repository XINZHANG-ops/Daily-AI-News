# Cursor

## What It Is
AI-powered coding IDE that pivoted to agent orchestration with Cursor 3. Competes directly with Claude Code and OpenAI Codex in the rapidly consolidating AI coding market. SpaceX invested $60B in Cursor, positioning it as a major independent player in the agentic coding space.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Cursor (independent) |
| Investment | $60B from SpaceX |
| Key Feature | Agent orchestration (Cursor 3) |
| Competitors | Claude Code, Codex, GitHub Copilot |

## Significance
Cursor represents the independent third player in the AI coding market, positioned between Anthropic's Claude Code and OpenAI's Codex. The $60B SpaceX investment signals that SpaceX sees AI coding tools as strategic infrastructure for its own operations — and possibly as a counterweight to Microsoft/GitHub's Copilot dominance. Cursor 3's pivot to agent orchestration puts it in direct competition with Symphony's vision of fleet-scale agent management. If Cursor can remain model-agnostic while Claude Code is tied to Anthropic models and Codex to OpenAI, it could become the Switzerland of AI coding tools.
