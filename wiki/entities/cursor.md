# Cursor

## What It Is
AI-powered coding IDE that pivoted to agent orchestration with Cursor 3. Competes directly with Claude Code and OpenAI Codex in the rapidly consolidating AI coding market. Acquired by SpaceX in a $60 billion stock deal (June 2026) to integrate advanced agentic coding into engineering and aerospace workflows. Cursor 3.0 launched with "Agents Window" — a full-screen workspace running multiple AI agents in parallel across local environments, isolated Git worktrees, SSH remotes, and cloud instances.

## Significance
Cursor represents the shift from "coding assistant" to "engineering infrastructure." The $60B acquisition by SpaceX is a massive bet on "Software-Defined Engineering," where agentic code is used to design, simulate, and optimize complex aerospace systems in a tight loop, potentially bypassing human bottlenecks. 

Cursor 3.0's "Agents Window" represents the shift from "one agent per tab" to "multi-agent workspace" — the transition from copilot to autonomous team. By supporting Git worktrees and SSH remotes, Cursor positions itself for distributed agent teams — a preview of how development environments will handle "agents everywhere" rather than "one agent per developer."

Cursor now competes with Windsurf 2.0's always-on cloud execution model. While Cursor emphasizes parallel workspaces across isolated environments, Windsurf emphasizes persistence and continuity. This reflects two different philosophies: workspace isolation vs. continuous execution.
