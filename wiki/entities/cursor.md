# Cursor

## What It Is
AI-powered coding IDE that pivoted to agent orchestration with Cursor 3. Competes directly with Claude Code and OpenAI Codex in the rapidly consolidating AI coding market. SpaceX invested $60B in Cursor, positioning it as a major independent player in the agentic coding space. Cursor 3.0 launched with "Agents Window" — a full-screen workspace running multiple AI agents in parallel across local environments, isolated Git worktrees, SSH remotes, and cloud instances.

## Significance
Cursor represents the independent third player in the AI coding market, positioned between Anthropic's Claude Code and OpenAI's Codex. The $60B SpaceX investment signals that SpaceX sees AI coding tools as strategic infrastructure. Cursor 3.0's "Agents Window" represents the shift from "one agent per tab" to "multi-agent workspace" — the transition from copilot to autonomous team. By supporting Git worktrees and SSH remotes, Cursor positions itself for distributed agent teams — a preview of how development environments will handle "agents everywhere" rather than "one agent per developer."

Cursor now competes with Windsurf 2.0's always-on cloud execution model. While Cursor emphasizes parallel workspaces across isolated environments, Windsurf emphasizes persistence and continuity. This reflects two different philosophies: workspace isolation vs. continuous execution.
