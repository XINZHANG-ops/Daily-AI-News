# CopilotKit

## What It Is
CopilotKit is a framework designed for "Generative UI," enabling AI agents to dynamically create and update UI components—such as buttons, charts, and forms—in real-time during a conversation, rather than just returning text or static blocks.

## Significance
CopilotKit represents a shift in the agentic UI paradigm: moving from "Chat-as-an-Interface" to "UI-as-a-Dynamic-Output." This allows agents to adapt the user's workspace based on the current task, significantly reducing the friction between LLM reasoning and human action.
