# GenClaw

## What It Is
Code-Driven Agentic Image Generation framework that explores using code as a controllable visual canvas before calling image generation models. Instead of direct text-to-image prompting, GenClaw treats visual composition as a programmable problem.

## Significance
GenClaw represents an emerging paradigm: using code (programmatic control structures, loops, conditionals) as the interface to image generation rather than natural language prompts. This positions AI image generation as a programmable medium rather than a prompting game.

For developers, this approach offers precise control over composition, layout, and iterative refinement — something that pure natural language prompting struggles to achieve. The "agentic" label suggests the system can autonomously decide how to translate code instructions into visual output.
