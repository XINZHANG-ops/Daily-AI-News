# Gemma 4

## What It Is
Gemma 4 is Google's most capable open model family, released April 2, 2026 under Apache 2.0 license — the first fully permissive Gemma release.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Google DeepMind |
| Release Date | April 2, 2026 |
| License | Apache 2.0 |
| Variants | 2B, 3B, 26B MoE, 31B dense |
| Context Window | 256K |
| Arena Ranking | #3 (31B), #6 (26B MoE) |

## Significance
Gemma 4's Apache 2.0 release marked a strategic shift — removing barriers to enterprise adoption while challenging Meta's Llama dominance. The 31B dense model ranked #3 globally on Arena AI's open leaderboard, while the 26B MoE (first Gemma MoE with 128 experts) ranked #6, outperforming models 20x their size.

With 400M+ downloads and 100K+ variants, Gemma has built a significant open-source ecosystem.
