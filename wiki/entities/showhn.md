# Show HN Gallery

## What It Is
Show HN Gallery (InsipidPoint/showhn) is an AI-powered visual gallery for Show HN projects, with automated screenshots, AI analysis, and full-text search. It gained 28 stars on GitHub on June 3, 2026.

## Significance
Show HN Gallery tackles a real Hacker News pain point: visual projects are buried under text-heavy links because HN's text-first format disadvantages them. The auto-screenshot + AI analysis pipeline flips this — every project gets a visual card, with the AI generating a description for ranking. This is content-curation-as-a-service, powered by computer vision and LLMs.

The 28 stars on day-one (notable for a curation project) suggests HN's visual-problem is a real, acknowledged issue. It also shows the AI-curation pattern maturing: the curation layer (AI) is now cheap enough to deploy on community-scale projects.
