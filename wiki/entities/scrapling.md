# Scrapling

## What It Is
Scrapling is an adaptive web scraping library that leverages AI to automatically re-locate elements and adapt to website structure shifts in real-time.

## Significance
It solves the "brittle selector" problem in web scraping. By using AI to understand the *semantic* meaning of an element rather than its *static* XPath or CSS selector, Scrapling allows scrapers to remain functional even after site redesigns.
