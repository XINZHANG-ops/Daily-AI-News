# Hatch

## What It Is
Hatch is Meta's OpenClaw-inspired AI assistant being built for Instagram to enable agentic shopping — allowing users to purchase items seen in Reels more easily. It represents Meta's answer to "shopping inside content" where users see an outfit in a Reel, tap once, and Hatch handles the entire purchase flow.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Meta |
| Platform | Instagram (Reels) |
| Function | Agentic shopping assistant |
| Target | In-content commerce |
| User Base | Instagram's 2B+ users |

## Significance
Hatch directly challenges Shopify's "agentic commerce" play and Amazon's 1P strategy. With Instagram's 2B+ users, Meta could become the largest "in-content" commerce platform before Amazon fully pivots to voice/agent shopping. This represents Meta's strategic response to the broader shift from search-based to agent-driven commerce.
