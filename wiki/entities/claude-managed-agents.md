# Claude Managed Agents

## What It Is
Claude Managed Agents is Anthropic's hosted multi-agent orchestration product, launched in public beta on May 30, 2026. It provides enterprises with sandboxed, managed infrastructure for running multiple AI agents that can collaborate on complex tasks.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Launch Date | 2026-05-30 |
| Type | Managed multi-agent infrastructure |
| Performance Improvement | 90.2% over single-agent setups |
| Key Features | Sandboxing, multi-agent orchestration |
| Early Customers | Notion, Rakuten, Asana |

## Significance

The 90.2% performance claim signals Anthropic's entry into the multi-agent infrastructure layer — the platform level where agent orchestration happens. This mirrors AWS's early compute services: providing the infrastructure on which others build applications.

The launch puts Anthropic in direct competition with OpenAI's enhanced Codex multi-agent features (released the same week). The agent orchestration layer is becoming a platform war — whoever controls the multi-agent infrastructure wins the enterprise AI relationship.

Early enterprise adoption from Notion, Rakuten, and Asana shows traction, but the real prize is becoming the default infrastructure layer for enterprise AI workflows.
