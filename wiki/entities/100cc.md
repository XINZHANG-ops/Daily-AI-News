# 100cc

## What It Is
100cc (yujqiao/100cc) is a minimal coding agent harness that reimplements Claude Code's core functionality in roughly 100 lines of code using the Vercel AI SDK. It gained 23 stars on GitHub on June 3, 2026.

## Significance
100cc embodies the "demystification" trend in AI coding tooling: rather than treating Claude Code as a black box, 100cc shows that the core agentic loop (prompt → tool call → result → loop) is achievable in 100 lines. This is the same spirit as the NanoClaw / 100-line movement in robotics — proving the abstraction is portable.

Its release coincides with the broader agent infrastructure convergence (Solara, Antigravity, MAI family), suggesting that the agentic loop itself is becoming a commoditized building block that anyone can deploy. The real differentiation moves up the stack: model quality, tool libraries, and UX.
