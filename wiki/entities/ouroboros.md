# ouroboros

## What It Is
ouroboros is an open-source "Agent OS" that shifts the agent paradigm from prompt engineering to specification-driven behavior. It provides a declarative operating system for AI agents with built-in planning and execution loops, enabling developers to define what agents should accomplish rather than how to prompt them into doing it. With 3,677 stars on GitHub, it represents a philosophical challenge to the dominant prompt-centric agent development model.

## Significance
ouroboros attacks one of the most painful aspects of current agent development: the fragility of prompt engineering. By providing a declarative layer where developers specify goals, constraints, and success criteria rather than crafting elaborate prompts, it aims to make agent behavior reproducible and maintainable. The "Stop prompting. Start specifying." tagline captures a growing consensus that the next phase of agent development requires software engineering discipline rather than linguistic artistry.
