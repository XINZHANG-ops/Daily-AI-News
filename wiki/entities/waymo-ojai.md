# Waymo Ojai

## What It Is
Waymo Ojai is the sixth-generation Waymo robotaxi, built by Geely (Zeekr platform). It represents a major cost reduction milestone for autonomous ride-hailing, featuring 42% fewer sensors than the previous generation while maintaining safety performance.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Generation | 6th |
| Platform | Geely Zeekr |
| Sensors | 13 cameras, 4 lidar, 6 radar |
| Sensor Reduction | 42% fewer than 5th generation |
| Vehicle Cost | $125,000 (vs $200,000 previous) |
| Base Vehicle Cost | $38,000 Zeekr (vs $75,000 Jaguar I-PACE) |
| Hardware Cost | ~$20,000 |
| Detection Range | 500m dark-range |
| Camera Resolution | 17 megapixel imagers |
| Expansion Markets | Denver, Las Vegas, San Diego |

## Significance

The Waymo Ojai is significant because it proves unit economics for robotaxis. The $75,000 cost reduction per vehicle across a fleet of thousands translates to hundreds of millions in savings. For the first time, Waymo is not just proving the technology works — it's proving the business model works.

The sensor reduction (42% fewer) while maintaining safety is enabled by:
- Higher resolution 17-megapixel cameras
- 500m dark-range detection capability
- Improved software fusion algorithms

This represents a shift from "sensor soup" to "smart sensing" — using better sensors with smarter software rather than more sensors with basic processing.
