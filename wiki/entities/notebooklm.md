# NotebookLM

## What It Is
Google's AI-powered research assistant notebook product, designed for grounding generative AI in user-uploaded sources. Originally launched as a notebook-style research tool integrated with Gemini, NotebookLM evolved into a major consumer distribution surface for Google's agentic stack. On June 8, 2026, NotebookLM upgraded to Gemini 3.5 + Antigravity integration, with each notebook getting its own secure cloud computer that can write and run code across 100+ curated skills.

## Significance
NotebookLM's Antigravity integration is the first major consumer distribution for Google's Antigravity agentic platform. By giving each notebook its own secure cloud computer with 100+ curated skills, NotebookLM becomes the canonical "personal agent computer" pattern — every user gets an isolated agentic environment for research, code, and analysis.

The Gemini 3.5 + Antigravity upgrade deepens Google's reach into the agent ecosystem and signals that the "personal agent computer" pattern is becoming a Google product category rather than an experimental feature. For enterprises evaluating agent platforms, NotebookLM demonstrates how Google's Antigravity can scale to consumer-grade simplicity while maintaining the security controls enterprises require.
