# ShowUI-Pi

## What It Is
A Vision-Language-Action (VLA) model specifically optimized for GUI control. Unlike traditional models that output discrete coordinates, ShowUI-Pi generates continuous cursor trajectories, enabling dexterous and precise actions such as dragging, rotating, and complex GUI interactions.

## Significance
ShowUI-Pi marks a transition from "AI that suggests" to "AI that operates." By mastering continuous motion in GUIs, it closes the gap between high-level intent and low-level execution, paving the way for agents that can use any software as a human would, without needing specialized APIs.
