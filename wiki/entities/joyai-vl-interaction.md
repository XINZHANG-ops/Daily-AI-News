# JoyAI-VL-Interaction

## What It Is
JoyAI-VL-Interaction is an 8B-scale vision-first interaction model designed for real-time video-language interaction. It enables AI to be "present" in a scene and autonomously decide when to speak based on visual triggers.

## Significance
Unlike traditional VLM (Vision-Language Model) interfaces that require a user to prompt the AI, JoyAI-VL-Interaction moves toward "proactive" visual agency. This is a key step toward embodied AI that can interact naturally with the physical world in real-time.
