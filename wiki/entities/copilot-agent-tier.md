# GitHub Copilot Agent Tier

## What It Is
GitHub's new Copilot Agent Tier pricing model announced April 29, 2026 — $49 per autonomous agent per month, with free-tier agent usage capped at 500 commits/month. The pricing is calibrated against human developer costs ($10K+/month fully loaded) to make it look cheap while capturing the surplus value that agentic coding creates.

## Significance
GitHub just solved its agent economics problem the way every platform eventually does: by charging for the resource that was previously free. The 500-commit free cap preserves the open-source illusion for hobbyists while forcing any serious agent deployment into paid tiers. This is a direct response to the April 27 revelation that GitHub Actions compute minutes hit 2.1 billion in a single week — the infrastructure economics were unsustainable. The secondary effect will be agent vendor consolidation: smaller agent frameworks that relied on GitHub's free tier now face a $49/month per agent tax, favoring well-funded players like Cursor, Claude Code, and Codex. GitHub is essentially becoming a tollbooth on the agent economy it helped create.
