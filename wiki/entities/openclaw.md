# OpenClaw

## What It Is
OpenClaw is an open-source autonomous agent framework that reached 302,000 stars as the fastest-growing open-source project in GitHub history.

## Key Facts

| Attribute | Value |
|-----------|-------|
| GitHub Stars | 302K+ |
| Models Supported | 75+ (Anthropic, OpenAI, Google, xAI, DeepSeek, Mistral, Ollama) |
| Key Feature | Zero subscription, API costs only |

## Significance
OpenClaw's explosive growth and Jensen Huang's designation as "the new computer" at GTC 2026 positioned it as a major open-source agent platform. NemoClaw was introduced as the enterprise stack for safe OpenClaw deployment.

On May 11, 2026, it was revealed that OpenClaw creator Peter Steinberger joined OpenAI in February 2026 after Meta failed to hire him. Meta had reportedly planned to use Steinberger's expertise for its Hatch agentic shopping tool, but his move to OpenAI forced Meta to switch plans. Jensen Huang called OpenClaw "the next ChatGPT" — a recognition that agentic interfaces represent the same platform shift that chat interfaces did in 2022. Steinberger's move to OpenAI consolidates the most important open-source agent talent under the company with the most compute and distribution.
