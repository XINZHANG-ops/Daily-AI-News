# Claude Haiku 4.5

## What It Is
Claude Haiku 4.5 is Anthropic's fast and cost-efficient model, added to GitHub Copilot Cloud Agent on May 18, 2026. It's positioned for simple coding tasks where full Opus or Sonnet capability is unnecessary — the "good enough" tier for high-volume, low-cost coding assistance.

## Significance
The addition of Haiku to GitHub Copilot is notable — Microsoft is hedging its OpenAI dependency while maintaining the partnership. This represents a strategic dual-vendor approach: Haiku for simple tasks, OpenAI's GPT models for complex reasoning.

The model tiering reflects the emerging reality in agentic AI: complex reasoning goes to premium models (Opus/Sonnet), while simple tasks go to fast/cheap models (Haiku/GPT-mini). Agents are increasingly routing based on task complexity — a pattern that mirrors traditional software where different tools handle different complexity levels.
