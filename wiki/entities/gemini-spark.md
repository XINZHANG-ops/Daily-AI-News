# Gemini Spark

## What It Is
Google's consumer AI agent for daily tasks. Released at Google I/O 2026 to compete directly with Anthropic's Claude Code and OpenAI's Operator in the emerging agent market.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Release Date | 2026-05-19 |
| Creator | Google |
| Type | Consumer AI agent |

## Significance
Gemini Spark positions Google directly against Anthropic's Claude Code and OpenAI's Operator in the consumer/agentic AI market. Unlike Claude Code which targets enterprise developers, Spark appears designed for general consumer use — managing daily tasks, schedules, and workflows. This consumer focus represents Google's attempt to own the "AI assistant" relationship that Apple Siri and Amazon Alexa never delivered on.
