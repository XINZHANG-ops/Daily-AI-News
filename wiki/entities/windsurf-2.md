# Windsurf 2.0

## What It Is
Codeium's AI-powered coding IDE that introduced "always-on" cloud execution with Devin Cloud integration. Users can dispatch tasks to Devin's cloud environment where agents continue running even after local devices shut down, bridging the gap between ephemeral local sessions and persistent cloud agents.

## Key Facts

| Attribute | Value |

| Release Date | 2026-05-22 |

| Differentiation | "Close your laptop, agent keeps running" |

## Significance
The "close your laptop, agent keeps running" capability addresses the biggest friction in autonomous workflows: user availability. This bridges the gap between temporary local sessions and persistent cloud agents—the holy grail for truly asynchronous development. It represents a shift from "tethered to desk" development (where agents stop when the computer shuts) to "always-on" development where code agents run continuously in the cloud.

Windsurf 2.0 competes directly with Cursor 3.0's multi-agent workspace approach. While Cursor emphasizes parallel execution across local environments, Git worktrees, and SSH remotes, Windsurf emphasizes persistence and continuity—agents that outlive the developer's active session. This reflects two different philosophies about agentic development: workspace isolation vs. continuous execution.
