# claw-code

## What It Is
claw-code is a clean-room Python (later Rust) implementation of Claude Code's agent harness architecture, built from scratch without copying proprietary code. It became the fastest project in GitHub history to reach 100,000 stars.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | ultraworkers |
| GitHub Stars | 181K+ |
| License | Clean-room (no proprietary code) |

## Significance
claw-code's explosive growth (100K stars in hours after Claude Code's source leak) demonstrated that closed-source agent architectures can be rapidly reconstructed. The incident raised fundamental questions about the durability of closed-source agent business models.

The ability for an AI to reconstruct a competitor's architecture in hours using clean-room techniques creates a new competitive reality: the harness layer is as vulnerable to replication as model weights.
