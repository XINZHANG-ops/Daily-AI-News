# Project Solara

## What It Is
Project Solara is Microsoft's "chip-to-cloud" platform for AI agent-first devices, announced at Build 2026. It represents a hardware ecosystem play featuring wearable badge and desk hub concepts powered by Qualcomm silicon, top-facing cameras, and 5G connectivity.

## Significance
Project Solara represents Microsoft's answer to the "post-app" computing vision — devices that run agents instead of apps. Unlike Apple's closed ecosystem or OpenAI's rumored hardware with Jony Ive, Microsoft is positioning Solara as an open reference design for hardware partners, with enterprise pilots in healthcare, retail, and hospitality.

The move directly challenges Google's Android dominance in the assistant-device market while hedging against a future where traditional apps become obsolete. By open-sourcing the reference design, Microsoft aims to become the Android of AI agents — owning the platform while letting partners build the hardware.
