# Stripe Agentic Commerce

## What It Is
Stripe's comprehensive agentic commerce platform announced at Stripe Sessions 2026, comprising 288 new products centered on enabling AI agents to transact autonomously. Core components include Link Wallets for Agents (one-time-use virtual cards that prevent AI agents from ever seeing raw payment credentials), Streaming Payments for real-time token billing on the Tempo blockchain, partnerships with Google AI Mode and Gemini, and expanded Radar fraud protection targeting AI-specific abuse.

## Significance
This is the infrastructural moment that makes autonomous AI commerce viable. Before Link Wallets, every AI agent would need to store or access real payment credentials — a non-starter for security teams. By wrapping one-time virtual cards around agent transactions, Stripe solved the authentication layer that was blocking agent-to-agent commerce. The partnership with Google AI Mode (joining existing OpenAI, Microsoft, and Meta integrations) means Stripe now powers checkout inside every major AI assistant. Streaming Payments on Tempo blockchain solve the billing-mismatch problem where agents burn tokens faster than monthly cycles can handle. This is Stripe positioning itself as the Visa of the agent economy.
