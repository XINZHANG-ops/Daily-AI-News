# Mercury Agent

## What It Is
Mercury Agent is a soul-driven AI agent with permission-hardened tools, token budgets, and multi-channel access. It runs 24/7 from CLI or Telegram, representing a new approach to personal AI agents that emphasizes persistent operation and diverse access channels.

## Significance
Mercury Agent represents a category of "personal AI agents" that differ from enterprise-focused platforms. Its emphasis on "permission-hardened tools" suggests a security-conscious design — agents that can only take actions within explicit permission boundaries. Token budgets address cost control, an increasingly important consideration as agent usage scales.

The 24/7 operation model via CLI or Telegram indicates a lightweight, always-available approach to agentic AI — different from browser-based or desktop agent interfaces. This could appeal to developers who want persistent background agents rather than on-demand interactions.
