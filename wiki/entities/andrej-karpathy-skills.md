# Andrej Karpathy Skills

## What It Is
A single CLAUDE.md file by forrestchang that enforces four principles to improve Claude Code behavior: think before coding, simplicity first, surgical changes, and goal-driven execution. Derived from Andrej Karpathy's observations on LLM coding pitfalls. 105K+ stars in 3 months, making it one of the fastest-growing AI-related repos.

## Significance
This repo represents a new category: "prompt engineering as configuration." Instead of building complex agent frameworks, it uses a single CLAUDE.md file to enforce behavioral principles. The 105K stars in 3 months demonstrates massive hunger for simple, principle-based approaches to taming LLM coding behavior. It validates Karpathy's thesis that many LLM coding pitfalls are behavioral, not capability problems. The four principles — think first, keep it simple, make surgical changes, stay goal-driven — are a direct response to Claude Code's documented quality regression issues (reduced reasoning, verbosity, caching bugs).
