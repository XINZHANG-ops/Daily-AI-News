# Antigravity Platform

## What It Is
Google's complete agent runtime platform launched May 2026 at Google I/O, designed for building, deploying, and managing AI agents at enterprise scale. Features privacy controls as a core capability, distinguishing it from consumer-focused agent products.

## Key Facts

| Attribute | Value |

| Key Feature | Privacy controls built in |
| Target | Enterprise adoption |

## Significance
The Antigravity platform is the real story behind Gemini Spark — it's a complete agent runtime with privacy controls, suggesting Google is targeting enterprise adoption more than consumers. This positions them differently from OpenAI's consumer-focused ChatGPT strategy and Anthropic's professional developer focus.

For enterprises, Antigravity offers a Google-native way to deploy agents that stay within Google's ecosystem while providing the governance and compliance controls enterprises require. The privacy controls address a key enterprise concern: keeping proprietary data within acceptable boundaries.
