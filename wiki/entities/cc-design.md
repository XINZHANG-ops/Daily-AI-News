# cc-design

## What It Is
cc-design is a high-fidelity HTML design and prototype guidance skill for AI agents. It provides structured methodologies for creating polished, production-quality web interfaces through AI coding agents. The skill targets the workflow gap between AI-generated "functional" web pages and truly refined design implementations.
