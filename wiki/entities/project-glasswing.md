# Project Glasswing

## What It Is
Project Glasswing is a restricted-access cybersecurity consortium launched by Anthropic to share the defensive insights derived from Claude Mythos discoveries. Rather than releasing Claude Mythos publicly, Anthropic assembled a coalition of 40+ technology and security companies who receive controlled access to the model's vulnerability discoveries.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Anthropic |
| Launch Date | April 2026 |
| Partners | Microsoft, Apple, Amazon, Google, NVIDIA, CrowdStrike, 35+ others |
| Commitment | $100M in usage credits to fix discovered vulnerabilities |
| Purpose | Defensive sharing of Claude Mythos cybersecurity insights |

## Significance
Project Glasswing represents a new model for AI safety — instead of releasing dangerous capabilities broadly, Anthropic created a closed consortium for responsible sharing. This approach acknowledges that Claude Mythos's vulnerability discovery capabilities are too valuable to waste (thousands of real vulnerabilities found) but too dangerous for open release.

The consortium approach has implications for how frontier AI capabilities are governed: government pressure (CISA discussions, emergency CEO calls) suggests that even restricted releases require coordination with national security interests.
