# Pixelle-Video

## What It Is
Pixelle-Video is an open-source AI-powered fully automated short video engine that handles scriptwriting, AI imagery generation, voice synthesis, background music, and final assembly from a single topic input. Released by AIDC-AI, it reached 10.5K stars on GitHub in May 2026.

## Key Facts

| Attribute | Value |

| Creator | AIDC-AI |
| Stars | 10.5K+ |
| Category | AI video generation |
| Pipeline | Scriptwriting, imagery, voice, music, assembly |

## Significance
Pixelle-Video represents the spread of agent frameworks beyond coding into creative domains. While Ruflo and GitNexus tackle coding agent infrastructure, Pixelle-Video applies the same autonomous pipeline philosophy to video production — a single topic triggers a multi-stage creative process without human intervention. This convergence signals that agent frameworks are becoming general-purpose creative infrastructure, not just developer tools.
