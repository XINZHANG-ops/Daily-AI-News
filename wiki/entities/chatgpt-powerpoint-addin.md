# ChatGPT PowerPoint Add-In

## What It Is
A beta Microsoft PowerPoint add-in released by OpenAI that embeds ChatGPT directly into the Microsoft Office productivity suite. Available for both free and paid ChatGPT users, the sidebar allows users to create and edit presentations from text prompts, files, or connected external accounts.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Release Date | 2026-05-24 (beta) |
| Creator | OpenAI |
| Platform | Microsoft PowerPoint |
| Pricing | Free and paid ChatGPT users |
| Data Sources | Gmail, Outlook, SharePoint |

## Features
- Create presentations from text prompts
- Edit existing slides via conversational interface
- Connect to Gmail, Outlook, and SharePoint for content sourcing
- File upload support for generating slides

## Significance
This marks a significant expansion of OpenAI's strategy beyond its own chat interface into ambient AI across enterprise tools. The deal is enabled by a non-exclusive 2032 license from the restructured Microsoft-OpenAI partnership — a notable shift from the previous exclusive arrangement. This puts ChatGPT directly inside Microsoft's productivity suite, competing head-on with Microsoft Copilot. It represents the first time Microsoft has explicitly opened the door to a third-party AI rival within its own ecosystem.
