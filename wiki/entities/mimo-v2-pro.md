# MiMo-V2-Pro

## What It Is
MiMo-V2-Pro is Xiaomi's flagship AI model, a trillion-parameter model with 42 billion active parameters. It was tested anonymously on OpenRouter as "Hunter Alpha" before formal reveal.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Xiaomi |
| Parameters | 1 trillion (42B active) |

| Benchmark | GDPval-AA Elo 1426 |

## Significance
MiMo-V2-Pro demonstrated the "stealth launch" pattern — anonymous testing on OpenRouter to gather unbiased benchmark data before brand expectations distorted assessments. It achieved #3 globally on agent benchmarks behind Claude Opus 4.6.

The model costs 67% less than Claude Sonnet 4.6 for agentic tasks, signaling the cost efficiency competition in AI.
