# last30days-skill

## What It Is
mvanhorn/last30days-skill is an AI agent skill that researches any topic across Reddit, X, YouTube, Hacker News, Polymarket, and the web, then synthesizes a grounded summary with citations. Trending as a top GitHub repo on June 9, 2026 (35.9k stars, 2.9k forks) — the most-starred trending repo of the day and one of three infrastructure-layer repos (last30days-skill, turbovec, google/skills) trending simultaneously.

## Significance
last30days-skill represents the commoditization of "grounded research" — a task that required human analysts, RAG pipelines, or specialized search APIs in 2024 is now a single skill that any compatible agent can load. The 35.9k stars signal that the developer community has voted: research synthesis is one of the most-wanted agent capabilities.

The "skill" framing is notable: last30days-skill is designed to be loaded into Claude Code, Cursor, or any MCP-compatible agent runtime, exactly the same way Google's google/skills targets Gemini agents. The "skill" abstraction is becoming a category of its own — discrete, loadable capability packs that any agent can use. The June 9 trending page showing both last30days-skill and google/skills in the top 3 is a category formation moment.
