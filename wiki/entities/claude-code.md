# Claude Code

## What It Is
Claude Code is Anthropic's CLI coding assistant that enables developers to build software faster by running agents that can browse, edit, and write code. It became the primary driver of Anthropic's revenue growth from $9B to $30B ARR.

## Significance
Claude Code's viral success in late 2025-early 2026 transformed Anthropic's business trajectory. The Super Bowl ad mocking ChatGPT's decision to show ads ("Claude will always be ad-free") drove consumer adoption. By March 2026, Claude subscriptions had more than doubled.

Claude Code was the subject of a major leak in early April when Anthropic accidentally exposed 512,000 lines of TypeScript source via misconfigured .map files in the npm package. This spawned the explosive growth of claw-code.

Claude Code Channels enables Telegram and Discord integration via MCP-based plugins. Claude Computer Use allows Claude to perform tasks on the user's behalf (file transfers, photo resizing).
