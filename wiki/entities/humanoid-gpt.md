# Humanoid-GPT

## What It Is
A repository by GalaxyGeneralRobotics implementing zero-shot motion tracking for humanoid robots. It utilizes a Causal Transformer trained on a massive dataset of 2 billion frames of motion data to translate instructions or targets into precise robotic movements.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Release Date | 2026-06 |
| Creator | GalaxyGeneralRobotics |
| Dataset | 2B frames of motion data |
| Architecture | Causal Transformer |

## Significance
Humanoid-GPT represents a leap in "Physical AI", demonstrating that the transformer architecture's ability to predict next-tokens can be applied to predicting next-motions at a scale that allows for zero-shot capabilities, reducing the need for tedious per-robot manual tuning.
