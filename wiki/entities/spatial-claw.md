# SpatialClaw

## What It Is
SpatialClaw is a training-free framework developed by NVlabs that utilizes code as the action interface for spatial reasoning. It allows AI agents to perform spatial tasks by generating and executing code, rather than relying on raw coordinate predictions or high-level descriptions.

## Significance
By leveraging code as an interface for spatial reasoning, SpatialClaw bypasses the need for extensive fine-tuning on spatial datasets, demonstrating that structured code can serve as a highly effective "common language" for bridging visual perception and spatial action.
