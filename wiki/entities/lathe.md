# Lathe

## What It Is
Lathe is an experimental tool designed to facilitate active learning in the era of AI coding. It generates hands-on tutorials that require users to manually type code, specifically designed to combat the "intellectual laziness" and skill atrophy associated with over-reliance on AI code generation.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Purpose | Active Learning / Skill Retention |
| Method | Required Manual Typing / Interactive Tutorials |
| Stars | 1.2k |

## Significance
Lathe addresses a critical psychological and educational side effect of the "AI coding shift": the loss of deep understanding when LLMs handle all the implementation details. By forcing "active" participation, it attempts to preserve the human developer's ability to reason and implement from first principles.
