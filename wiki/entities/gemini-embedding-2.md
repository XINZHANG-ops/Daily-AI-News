# Gemini Embedding 2

## What It Is
Gemini Embedding 2 is Google's native multimodal embedding model supporting text, image, video, audio, documents, and code for AI search applications. Released May 29, 2026, it represents a significant advance over text-only embedding models by operating in a unified embedding space across modalities.

## Key Facts

| Attribute | Value |
|-----------|-------|
| Creator | Google DeepMind |
| Release Date | 2026-05-29 |
| Supported Modalities | Text, image, video, audio, documents, code |
| Primary Use | AI search, multimodal RAG |

## Significance
Gemini Embedding 2's native multimodal support (not just image+text) is the key differentiator — most embedding models handle text well but struggle with video and audio. This positions Google against OpenAI's text-embeddings and Cohere's multilingual offerings.

The critical implication: unified embedding space across modalities means semantic search across a PDF, its screenshots, and a related podcast becomes possible in a single query. This is the infrastructure layer that makes multimodal RAG practical — and Google's way of owning the retrieval layer regardless of which generative model users choose.
