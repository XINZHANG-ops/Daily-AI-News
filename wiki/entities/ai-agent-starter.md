# AI Agent Starter

## What It Is
A multi-provider AI agent framework supporting OpenAI, Anthropic, and Ollama with function calling, streaming, and conversation memory capabilities.

## Significance
AI Agent Starter provides a unified interface across three major LLM providers, enabling developers to switch between models without changing application code. The function calling and conversation memory features support building stateful agentic applications.
