# Mozilla

## Overview
Mozilla is an open-source software community best known for the Firefox web browser. In April 2026, Mozilla launched Thunderbolt — an enterprise self-hosted AI client — marking its entry into the enterprise AI infrastructure space.

## Key Relationships
Thunderbolt represents Mozilla's play in the self-hosted AI space — competing with cloud-based AI clients by offering data privacy and vendor independence for enterprises. The May 8 Firefox security hardening using Claude Mythos Preview marks Mozilla's entry into the AI-security tooling space, demonstrating that even browser vendors now trust AI models with production code security auditing.
